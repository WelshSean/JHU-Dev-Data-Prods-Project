<section class="slide " id="slide-1">
<h2>Overview</h2>
<ul>
<li>mtcars is one of the test datasets that can be loaded into R</li>
<li>It was extracted from the 1974 Motor trend US Magazine</li>
<li>A useful, manageable dataset for exploration</li>
<li>This application sets out to allow the user to explore possible Linear Models for the fuel efficiency and its dependencies on the other variables in the mtcars dataset.</li>
<li>The code for this presentation can be found on <a href="https://github.com/WelshSean/JHU-Dev-Data-Prods-Project/blob/master/Modelling_mtcars_deck/index.Rmd">Github</a></li>
<li>The R code for the Shiny application is also available in the same <a href="https://github.com/WelshSean/JHU-Dev-Data-Prods-Project">Repo</a></li>
</ul>
</section>

<section class="slide " id="slide-3">
<h2>The Problem that this application solves</h2>
<p>The application solves the linear model of the form shown below using the lm function in R. </p>
<p>\[
y_{i} = \beta_{0} + \beta_{1}\phi_{i}(X_{i}) \cdots + \beta_{n}\phi_{n}(X_{n}) + \epsilon
\]</p>
<p>The models that can be solved are at present limited to those of the form \(\phi_{i}(X_{i})= X_{i}\)</p>
<p>The tickboxes can be used to build formula that the lm function expects of the form y ~ a+b+c in order to tell R which covariates to include in the model. </p>
<p>The application returns the values for \(\beta_{0}, \beta{1} \cdots \beta_{n}\) to the user.</p>
<p>This allows the user to use the model to make predictions if they so wish</p>
<p>The application also returns the <a href="http://sphweb.bumc.bu.edu/otlt/MPH-Modules/BS/R/R5_Correlation-Regression/R5_Correlation-Regression7.html">Standard R Diagnostics</a> plot to the user in order to allow them to judge the validity of the model if they know enough about Linear Models to intepret the plot. </p>
</section>
<section class="slide " id="slide-4">
<h2>Why build this Application?</h2>
<ul>
<li>The application could serve as a useful learning aid for statistics students studying Linear Models</li>
<li>Provides a convenient way to play with a linear model for people who are interested</li>
<li>Provides a Minimal Viable Product(MVP) from which we could iterate to build a much more fully featured, useful application.</li>
</ul>
</section>
<section class="slide " id="slide-5">
<h2>Further work</h2>
<ul>
<li>Ability to select other models</li>
<li>Ability to toggle verbose output to see errors, p-values, etc</li>
<li>Ability to use model to carry out predictions.</li>
<li>Ability to evaluate Linear Models other than the case \(\phi_{i}(X_{i})\) other than \(\phi_{i}(X_{i})= X_{i}\)</li>
</ul>
</section>
